# Алфавиты и строки

* Алфавит - множество символов {a, b, c}
* Строка - последовательность символов w = {a, a, c, b}
* w = aacb, z = ba, wc = aacbba
* Степень w^3 = www, w^0 = ^
* Если есть конкатенация, алфавит, и нет других ограничение - это свободная группа [cтрок] над алфавитом.

# Формальные языки

Языком над данным алфавитом называется множеством строк.

**Формальный язык** — это набор слов (или строк), построенных по строгим, заранее заданным правилам. Эти правила определяют, какие именно последовательности символов считаются "правильными" (принадлежат языку), а какие — нет.

1. **Алфавит (Σ)** — конечный набор допустимых символов. Например:
   * `{0, 1}` для двоичных чисел.
   * `{a, b, c, ..., z}` для строчных латинских букв.
   * `{if, then, else, x, y, z, =, +, -, (, )}` для простого языка программирования.
2. **Слово (или строка)** — любая конечная последовательность символов из алфавита. Например, `"01011"` или `"hello"`.
3. **Язык (L)** — множество слов над данным алфавитом, которые удовлетворяют определенным правилам. Язык может быть:
   * **Конечным:** `{ "a", "ab", "ba" }`
   * **Бесконечным, но описываемым правилами:** все двоичные числа, кратные 3.

# Хомский

Классификация Хомского — это формальное разделение языков и грамматик по мощности и правилам, которые они допускают. Используется в теории формальных языков, компиляторах и языкознании. Всего четыре типа.

**Тип 0 — неограниченные грамматики (рекурсивно перечислимые языки).**
Правила без ограничений. Любое количество символов слева и справа.
Сила: наиболее мощные.
Реализация: машины Тьюринга.

**Тип 1 — контекстно-зависимые грамматики (КЗ).**
Правило имеет вид αAβ → αγβ, то есть длина строки не сокращается (|γ| ≥ 1).
Сила: описывают языки, которые требуют учета контекста.
Реализация: линейно-ограниченные автоматы.

**Тип 2 — контекстно-свободные грамматики (КС).**
Правила вида A → γ, слева всегда один нетерминал.
Самый важный класс для программирования.
Служат основой для обычных синтаксических деревьев.
Реализация: pushdown automata (автоматы со стеком).

**Тип 3 — регулярные грамматики.**
Правила вида A → aB или A → a, то есть символ + опциональный нетерминал.
Самые простые.
Описывают регулярные языки.
Реализация: конечные автоматы.

**Иерархия по мощности:**
Тип 3 ⊂ Тип 2 ⊂ Тип 1 ⊂ Тип 0.
